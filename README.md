# Nest.js template

## Packages installation

```bash
$ yarn
```

## Running the app

```bash
# development mode
$ yarn start:dev

# debug mode
$ yarn start:debug

# production mode
$ yarn start:prod
```

## Testing the app

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Building the app

```bash
# build to /dist folder
$ yarn build
```

## Checking for the code style exceptions

```bash
# eslint check
$ yarn lint:check
```
